# KUBO Sandbox

KUBO is a 2D sandbox building game made with HTML5 and JavaScript.  
Build worlds, place blocks, spawn animals, trigger disasters, and experiment freely in an interactive pixel environment.

## Play the game
https://ikx337.github.io/kubo-sandbox/

## Features
- Block building system
- Animals and entities
- Natural disasters (tornado, meteor, lightning, etc.)
- Explosives and items
- Physics and environment interactions
- Water, lava and survival mechanics

## Controls
Move: A / D  
Jump: Space  
Run: Shift  
Build / Place: Left Click  
Remove blocks: Right Click  

## Built With
- HTML5
- JavaScript
- Canvas API

## Project Goal
This project is an experimental sandbox game designed to explore physics interactions, world simulation, and creative gameplay systems directly in the browser.
